# LangChain Kurulumu

LangChain paketini kurmak için:

```bash
pip install -U langchain
# Python 3.10+ gerektirir
```

LangChain, yüzlerce LLM entegrasyonu ve binlerce başka entegrasyon sunar. Bunlar bağımsız sağlayıcı paketlerinde bulunur.

```bash
# OpenAI entegrasyonunu kurma
pip install -U langchain-openai

# Anthropic entegrasyonunu kurma
pip install -U langchain-anthropic
```

Kullanılabilir entegrasyonların tam listesi için Integrations sekmesine bakın.

LangChain'i kurduğunuza göre artık Hızlı Başlangıç rehberini takip ederek başlayabilirsiniz.
